Title: Ainkurunooru_vellankurugupathu_translation

Project Overview: 

This repository contains the digital translation assets for 10 selected songs (151-160)from the classical Tamil anthology Ainkurunooru. This project is part of a research thesis focusing on the application of Computer-Assisted Translation (CAT) tools in translating Sangam literature.
Tamil source text and English translations aligned at the segment level. Available in .tmx (Translation Memory eXchange) and .csv formats.

Tool Used: Created and managed using OmegaT.

Linguistic Focus: Addresses challenges in Machine Translation (MT) such as cultural-specific terminology and tamil grammar.
